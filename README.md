# PrimeFlix

Aplicação de filmes e séries desenvolvida em React utilizando a API do TMDB.

Neste projeto aprendi a consumir Apis, salvar dados no localStorage e organizar components reutílizaveis

![Tela do PrimeFlix 1](<src/readme-assents/detalhesfilme.png>)

![Tela do PrimeFlix 2](<src/readme-assents/detalhesfilme1.png>)

## 🚀 Funcionalidades

- Filmes em alta
- Séries populares
- Página de detalhes
- Página Favoritos com localStorage
- Responsividade

## 🛠 Tecnologias

- React
- Styled Components
- Axios
- Swiper
- React Router DOM

## ▶ Como executar

```bash
git clone + url 
npm install
npm run dev
